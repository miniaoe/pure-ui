### 项目介绍

本项目目的为学习 Vue3 构建的轻量 UI 框架，使用 `vite2⚡` + `vue3` + `scss` 可以快速了解并学习 Vue3 的新特性。

⚠️ **不建议应用 Pure UI 至生产环境**
